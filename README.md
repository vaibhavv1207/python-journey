## 💡 Java → Python: Key Differences I'm Learning

```java
// Java
int x = 5;
System.out.println("Hello " + x);
for (int i = 0; i < 5; i++) { }
```

```python
# Python — cleaner, no types, no semicolons
x = 5
print(f"Hello {x}")
for i in range(5):
    pass
```
